def addto(start, end):
定義了一個函式 addto，它接受兩個參數：

  • start：總和的起始值。
  • end：總和的結束值。
  • range(start, end + 1)
    生成從 start 到 end 的一系列整數（包含 end，因為 range 的結束值是開區間，需加 1）。
  • sum(...)
    使用 Python 內建的 sum 函數計算範圍內所有數字的總和。
  • return total
    將計算的總和值回傳給呼叫此函式的地方。

這段程式碼會計算 1+2+3+⋯+1001 + 2 + 3 + \cdots + 1001+2+3+⋯+100 的總和。數學公式 總和=n(n+1)2\text{總和} = \frac{n(n+1)}{2}總和=2n(n+1)，當 n=100n = 100n=100 時，總和為：

總和=100×(100+1)2=5050\text{總和} = \frac{100 \times (100 + 1)}{2} = 5050總和=2100×(100+1)=5050

題目4:
add_avoid_8

  • def add_avoid_8(start, end):
    定義函式，接受兩個參數：

    • start: 計算的起始值。
    • end: 計算的結束值。

  • valid_numbers
    使用列表生成式篩選符合條件的數字：

    • range(start, end + 1)
      生成從 start 到 end 的整數範圍。
    • '8' not in str(x)
      將數字轉換為字串，檢查是否包含字符 '8'，如果包含則排除。
    • x % 2 == 0
      篩選偶數，因為偶數的條件是可以被 2 整除（餘數為 0）。

  • print("符合條件的數字:", valid_numbers)
    將篩選出的符合條件的數字打印出來，方便檢查過程。

  • return sum(valid_numbers)
    將篩選出的數字累加並回傳結果。

  • add_avoid_8(1, 100)
    計算從 1 到 100 範圍內，避開包含數字 '8' 和基數的總和。

  • result
    儲存函式的回傳值（符合條件的數字總和）。

  • 打印結果： 使用 f-string 將結果插入到字串中，並打印到螢幕上。
